a responsive website that will display images descriptions and links 
to each of the portfolio projects .

Udacity Portfolio project for the Full Stack Web Developer Nanodegree.

# how to run it ?

	1- download source code (zip file)
	2- unzip the file 
	3- open the folder named "Portfolio_Project" 
	4- you will find a html file named "HTML"
	5- click right on it a choose "open with" your web browser 
	6- it will open in your web browser
